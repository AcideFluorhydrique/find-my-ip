Privacy Policy

Find My IP does not collect, store, or share any personal or sensitive user data externally, except as described below.

Data Processing

Find My IP uses the ipify API to determine the device's public IP address. This involves transmitting the device's IP address to ipify's servers. According to ipify, they do not store or track any data related to these requests. For more information, please refer to ipify's website.

Local Data Storage

Find My IP stores address history locally on the user's device to provide functionality within the app. This data is not transmitted to any external servers or third parties and remains solely on the user's device.

Consent

By using Find My IP, you consent to the processing of your data as described in this privacy policy.
